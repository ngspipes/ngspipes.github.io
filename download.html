We have extensive help information about all NGSPipes framework components, namely:

  • NGSPipes Domain Specific language. Documentation about all the language primitives and expressiveness can be found in the dsl wiki.

  • NGSPipes Tools Repository. Documentation about all the metadata primitives and expressiveness for adding new tools to an existing tools' repository or creating a new tools repository can be found in the tools wiki.

  • NGSPipes Engine . Documentation about engine for executing pipelines defined with the NGSPipes DSL can be found in the engine wiki.

  • NGSPipes Editor . The documentation about the user-friendly editor for defining pipeline can be found in the the editor wiki. However, pipelines can be defined using a simple text editor and, in this case, the documentation to consult should be the dsl wiki.

    • For executing pipelines, please download and install the NGSPipes Engine.

    For defining pipelines using NGSPipes Editor, please download and execute the NGSPipes Editor.
